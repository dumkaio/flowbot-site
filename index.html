We created flowbot for our purposes. However, we considered it might be useful for other Webflowers. You can do quick changes on the go, check items progress/status, sketch your ideas as a draft, or even publish. This is a beta version, so, test it out and feel free to mention @dmk_flowbot on Twitter!

1

Add a project

Add your Webflow project. Generate a token in the project settings -> integrations -> API access and just paste it to the flowbot.

2

Select a collection

Select one of the available CMS collections to manage. ℹ️ Creation a new collection is available in Webflow CMS only.

3

Manage collection items

Create, modify and delete items in your CMS collections.

Select existing or create new items

Pick an item and see its status

Manage the collection item

Edit necessary fields and publish

Supported actions

Create

Create new collection items from your phone.

Edit

Edit any CMS item in the project collections.

Save

Save items without publishing.

Publish

Publish single CMS item.  No need to publish the entire website.

Stage for Publish

Stage for publish your drafts.

Save as Draft

Save your items as a draft and proceed editing anytime.

Archive

Archive unnecessary items.

Unarchive

Unarchive items you need.

Duplicate

Speed up creating process by duplicating items.

Delete

Permanently delete items.
